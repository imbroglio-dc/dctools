# dctools: Reusable Utilities and Project Scaffolding for Biostatistics Research

A personal toolkit of repeated-task helpers for prediction- and
causal-inference research in R: project scaffolding (create_project()),
package and parallel-controller management, data validation, PHI-safe
output handling, clinical formulas, and house plotting/table defaults.
Data manipulation follows a dplyr-first grammar; large-data backends
(arrow, dbplyr) reuse the same verbs.
