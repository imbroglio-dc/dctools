# dctools

`dctools` is a personal toolkit of repeated-task helpers for prediction-
and causal-inference research in R. The package lives in the `helpers`
repository (the package itself is named `dctools`).

## Installation

``` r


# install.packages("pak")
pak::pak("imbroglio-dc/helpers")
```

## Scaffolding a new project

[`create_project()`](https://imbroglio-dc.github.io/helpers/reference/create_project.md)
is the single entry point for both project archetypes:

``` r


# Analysis project: clones the analysis template, stamps the name,
# optionally adds a targets pipeline.
dctools::create_project("sepsis-aki", type = "analysis", targets = TRUE)

# R package: wraps usethis::create_package() and layers the best-practice set
# (MIT license, testthat 3, roxygen markdown, pkgdown, CI, NEWS, .claude/).
dctools::create_project("mypkg", type = "package")
```

## Conventions

Data manipulation uses a **dplyr** surface grammar; for large data the
same verbs run over **arrow** or **dbplyr** backends. `data.table` is
reserved for profiled hot paths. See `CONTRIBUTING.md`.
